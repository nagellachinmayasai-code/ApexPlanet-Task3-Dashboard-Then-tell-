# ApexPlanet Task 3 - Sales Dashboard

This project is part of the ApexPlanet Data Analytics Internship.

## 📊 Project Overview
An interactive sales dashboard built using Excel to analyze business performance.

## 🔹 Features
- KPI metrics (Revenue, Orders, Avg Order Value, Top Category)
- Charts for Category, City, Product, and Payment analysis
- Interactive slicers for filtering data
- Key business insights

## 📈 Insights
- Electronics generates highest revenue
- Hyderabad leads in sales
- UPI is most used payment mode

## 🛠 Tools Used
- Excel
- SQL (for data cleaning)

## 📂 Files Included
- task3_dashboard.xlsx
- cleaned_sales.csv
